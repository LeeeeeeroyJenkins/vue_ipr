# Wishlist App

Приложение для создания и управления вишлистами.  
Позволяет создавать вишлисты с темами, добавлять элементы, редактировать их, менять визуальное оформление и хранить данные в localStorage.

---

## Функционал

- Создание вишлиста с выбором темы (`default`, `birthday`, `newyear`) и соответствующей иконкой.
- Редактирование вишлиста:
  - Название (inline-edit)
  - Цвет фона
  - Иконка
  - Обложка (URL)
- CRUD для элементов вишлиста:
  - Название, описание, ссылка, цена
  - Добавление изображения через файл (base64)
  - Пометка купленного товара
  - Удаление элемента
  - Drag & Drop для сортировки
- Поиск по названиям вишлистов и элементам с debounce
- Сохранение состояния в localStorage
- Модальные окна для добавления новых элементов/вишлистов
- Анимации для карточек и переходов между страницами

---

## Установка

1. Клонируйте репозиторий:

git clone https://github.com/LeeeeeeroyJenkins/vue_ipr.git
cd wishlist-app

2. Установите зависимости:

npm install

3. Запустите в dev-режиме:

npm run dev
